# Speaker Photos Directory

This directory contains photos of the panelists and moderators for the GLORY Sports Summit 2025.

## Photo Requirements

- **Format**: JPG or PNG preferred
- **Size**: Minimum 300x300 pixels, maximum 1000x1000 pixels
- **Aspect Ratio**: Square (1:1) for best results
- **File Naming**: Use lowercase with hyphens, e.g., `jesse-marsch.jpg`

## How Photos Are Used

1. Photos are automatically displayed in speaker cards when available
2. If a photo is missing or fails to load, the system displays attractive initials with color-coded backgrounds
3. All photos are responsive and optimized for mobile viewing
4. Photos appear in both collapsed and expanded panel views

## Adding New Photos

1. Add the photo file to this directory using the exact filename from the list above
2. The photo will automatically appear in the app without any code changes
3. Ensure photos are properly compressed and optimized for web use

## Fallback System

When photos are not available, the app generates:
- Color-coded circular avatars with speaker initials
- Consistent colors based on the speaker's name
- Professional appearance that matches the overall design
